<!-- Describe your project in brief -->

Ryze 是一个Java语言开发的测试框架，适用于HTTP(S)、Dubbo、JDBC、Redis等协议或工具的测试。

**核心特性**

1.测试用例与代码分离，采用JSON描述测试场景，保障测试用例描述的统一性和可维护性；

2.丰富的测试组件，轻松实现动态参数及复杂业务；

3.丰富的校验机制，灵活配置校验规则；

4.极强的可扩展性，满足个性化扩展需求；

5.简洁美观的测试报告；

